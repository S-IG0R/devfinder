# DevFinder

**DevFinder** - это React-приложение для поиска пользователей GitHub и отображения их профилей в виде наглядной карточки с ключевой информацией.

## 🚀 Функциональность

* 🔍 Поиск пользователей GitHub по никнейму
* 📇 Отображение данных пользователя в виде карточки
* ⌨️ Кастомный эффект "печатной машинки" - данные появляются постепенно, создавая ощущение динамического заполнения
* 🌗 Переключение светлой / тёмной темы
* 💾 Сохранение выбранной темы в `localStorage`
* 📱 Полностью адаптивная верстка (responsive design)
* 🔗 Работа с URL-параметрами поиска через `useSearchParams`
* ♻️ Использование кастомных React-хуков

## 🛠️ Технологии

* **React**
* **JavaScript + TypeScript**
* **Sass (SCSS)**
* **TanStack Query** - для работы с асинхронными запросами (использована впервые в рамках проекта)
* **React Router (useSearchParams) -** управление параметрами URL

## 🧠 Особенности реализации

### Работа с API

Для получения данных используется GitHub API. Логика запросов и кеширования реализована через **TanStack Query**, что позволяет:

* минимизировать количество запросов
* удобно управлять состояниями загрузки и ошибок
* кешировать данные пользователей

### Поиск и URL

Поисковый запрос синхронизируется с URL через `useSearchParams`, благодаря чему:

* можно делиться ссылкой с результатами поиска
* сохраняется состояние при перезагрузке страницы

### Темизация

Реализована система тем:

* переключение между светлой и тёмной темами
* сохранение выбора пользователя в `localStorage`
* применение темы при загрузке приложения

### Эффект печатной машинки

Кастомная логика, создающая эффект постепенного вывода текста:

* имитирует "ввод" данных в карточке
* реализована через кастомные хуки и управление состоянием
* поддерживает последовательную отрисовку 

### Архитектура

* Разделение логики и UI
* Переиспользуемые компоненты
* Кастомные хуки для инкапсуляции логики

## 📦 Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/your-username/devfinder.git

# Перейти в директорию проекта
cd devfinder

# Установить зависимости
npm install

# Запустить проект
npm run dev
```
